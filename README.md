# AI-resume-analyzer
AI Resume Analyzer is an intelligent web application that analyzes resumes using Generative AI and Langchain. The system evaluates uploaded resumes against job descriptions, extracts important information, identifies missing skills, and provides AI-powered suggestions to improve resume quality and ATS compatibility.

## Features

- Upload Resume (PDF/DOCX)
- AI-based Resume Analysis
- ATS Score Calculation
- Job Description Matching
- Skill Extraction
- Missing Skills Detection
- Resume Improvement Suggestions
- Interactive Streamlit Interface
- Fast and User-Friendly
